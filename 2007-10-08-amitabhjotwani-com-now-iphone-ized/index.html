ajot.me now iPhone-ized

Amit Jotwani

2007-10-08

As a proud iPhone owner, it was probably just a matter of time before ajot.me became iPhone friendly. I am happy to announce that it now is.

The website now automatically detects the iPhone/iPod touch's User Agent and serves up the content with the special theme only to iPhone and iPod touch visitors which is optimized for viewing on Apple's iPhone and iPod touch.

The website is quicker to load on the iPhone and displays an optimized version of my blog, formatted perfectly to fit the iPhone/iPod touch's screen - no zooming in to read the content is necessary. Ain't that awesome - iPhone awesome!!

(PS: For those interested, it was the magic of a wordpress plugin called iwphone, not much rocket science actually!!)
