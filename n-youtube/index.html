How I consume long form educational YouTube Videos

  • Download the video using Downie (available through SetApp)
  • Open the video in QuickTime Player, and export the file as audio
  • Save this file in Dropbox, which is already connected to Otter
  • Otter picks this up automatically, and transcribes it.
  • Grab coffee.
  • Now, watch the video with the transcript on the side, so I can highlight and note the things that resonate with me.
